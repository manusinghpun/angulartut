## Angular installation
> Install node js
> npm install -g @angular/cli
> npm version //to check version
> ng new blog //to create new angular project
> ng serve
> `http://localhost:4200`


## create component
> ng g c userList

## create module and its component
> ng g m users
> ng g c users/login
